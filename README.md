# eWave Studio
NSS Capstone Project implementing Javascript & Angular

## Usage
Sign up to create a user. There are 3 page views after logging in:
1. Live audio input (plugged in to computer via USB device)
2. MIDI triggers
3. Sequencer / Drum Machine
